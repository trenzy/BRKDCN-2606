Role Name
=========

Tenant role to create an ACI Tenant, Application Profile, EPGs, Bridge Domains, and other ACI objects under a Tenant. The `L3Out` role takes care of configuring the L3Out in the Tenant and the `static_path` role takes care of configuring static path findings in the EPG this role creates.
